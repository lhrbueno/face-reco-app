# Face-reco-app
REST API for **[face-reco-app-ui](https://github.com/lhrbueno/face-reco-app-ui)**

### Technologies
* ***express*** as server
* ***knexjs*** for persistence and ***PostgreSQL*** as database

### Configuration

##### .env.development.local
* The server port
* Database connection

## How to run it

1. Clone this repo
2. Run npm install
3. Run npm start
